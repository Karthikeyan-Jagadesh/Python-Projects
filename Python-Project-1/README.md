# Google Image Downloader 

This project is a **Python-based Image Downloader** that uses the  
[Google Custom Search API](https://developers.google.com/custom-search/v1/overview) to fetch and save images.  
The script allows you to enter a keyword and the number of images you want, then downloads them automatically.

---
## 📂 Project Structure
- `prgm.py` → Main Python script for downloading images
- `project1/` → Folder where downloaded images are saved
- `README.md` → Project documentation
---
## 🌟 Features
- Search and download images using the Google Custom Search API  
- Save images in a local folder (`project1/`)  
- Handles invalid or blocked image links gracefully  
- Easy to customize (change folder, file format, or number of results)
